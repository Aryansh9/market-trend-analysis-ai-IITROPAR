# Market & Trend Analysis AI

AI-powered stock market trend analysis and prediction system built using Python, Machine Learning, NLP, and Streamlit.

## Features

* Stock market trend analysis
* Price prediction using ML
* Data visualization
* Interactive dashboard
* Automated analysis

## Tech Stack

* Python
* Pandas
* NumPy
* Scikit-learn
* Streamlit
* Matplotlib

## Installation

```bash id="r8n1pq"
pip install -r requirements.txt
streamlit run app.py
```
